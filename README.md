# gcse-specifiactions
A json version of the gcse exam specifications.

To be clear, this is not the actual specifications, but a json version of the specifications. The actual specifications can be found on the exam board websites listed below:
- [AQA](https://www.aqa.org.uk/)

Also, I used LLMs to generate the json as it is impractical to write it out myself. I have done my best to ensure that the json is accurate, but there may be some errors. If you find any errors, please let me know in the issues tab of this repo and I will do my best to fix them - or make a PR yourself.
